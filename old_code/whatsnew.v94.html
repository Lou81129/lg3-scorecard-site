A new game type Individual Stableford has been added to the Game dropdown. When selected, Rows 5–8 automatically populate with each player's name followed by POINTS:

Scramble Team Card — Back Panel Rewrite (v94.6)

The back panel of the Scramble Team Card has been completely rewritten. All three sections are now Scramble-specific.

Section 1 — LG3 Local Rules (7 rules)

  • Root Relief — free relief if roots interfere with stance or swing
  • Wildlife Relief — free relief from gators, snakes, or protected wildlife
  • Embedded Ball Relief — through the green
  • Cart Path Relief — standard USGA relief
  • Safety Relief — nearest safe point, no penalty
  • Gimmies — inside the leather (33–34 inches)
  • Hole 18 Water Hazard — designated drop area

Section 2 — Scramble Rules (Team Format) (12 rules)

  • Four player USGA Scramble format
  • All players tee off → select best shot → all play from that spot
  • One team score per hole (Gross & Net)
  • Team Handicap uses USGA recommended method
  • Minimum 3 tee shots per player
  • Place ball within 1 club length (same cut)
  • On green: place within 6 inches
  • In bunkers: rake and place
  • Penalty areas: all players drop in same relief area
  • Lost ball / OB: proceed under USGA rules
  • No strokes after ball is holed
  • Scramble rounds not posted to GHIN

Section 3 — Weekly Game Notes (2 items)

  • Weather, pace, rulings, and tie breaks at LG3 Committee discretion
  • Have fun, play fast, enjoy the chaos

Print Both — Fully Redesigned (v94.7 · v94.8 · v94.9 · v94.10)

The Print Both workflow now produces a 2-page duplex PDF:

  • Page 1: Card 1 (top slot) + Card 2 (bottom slot)
  • Page 2: Shared back panel

Workflow:

  • Build Card 1 → Save
  • Build Card 2 → Save
  • Print Both → duplex PDF

Print Sizing Fix — Canon Printer (v94.10)

Card height inside PDF set to 3.75 inches with 0.25-inch inset. Prevents perforation overlap and ensures clean duplex output.

UI Fixes (v94.3 · v94.4)

  • Team Card Border: Full-height border restored
  • Format Toggle Spacing: Buttons now visually separated
