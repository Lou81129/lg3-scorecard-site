What’s New – Build 93

Released: June 23, 2026 · Built from v92 baseline

New Feature 1 — Individual 11s Game Profile

A new game type Individual 11s has been added to the Game dropdown. When selected, Rows 5–8 automatically populate with each player's name followed by 11 +/-:

RowLabel Example
Row 5Brian 11 +/-
Row 6Bob 11 +/-
Row 7Patrick 11 +/-
Row 8Martin 11 +/-
  • Static labels only — no scoring logic
  • Blank until a name is entered
  • Updates live as names are typed
  • Each card resolves its own names independently
  • Prints correctly in Color and B&W
  • Rows 1–4 unchanged

New Feature 2 — Individual 13s Game Profile

Identical to Individual 11s, but with 13 +/-:
